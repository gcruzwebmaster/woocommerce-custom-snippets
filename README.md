# WooCommerce Custom Snippets

A personal library of WooCommerce customization snippets 
collected across years of client eCommerce projects.

These are battle-tested fixes and enhancements — 
things that come up repeatedly in real builds.

---

## Categories

### 🛒 Checkout Customizations
- Add/remove/reorder checkout fields
- Custom field validation logic
- Conditionally show fields based on cart contents

### 📦 Order Management
- Auto-complete orders for virtual/downloadable products
- Custom order statuses with label and color
- Add custom data to order emails

### 💰 Pricing & Discounts
- Role-based pricing display
- Minimum order amount notices
- Custom discount logic without plugins

### 📋 Product Page Tweaks
- Reorder product page elements via hooks
- Custom tabs with dynamic content
- Hide/show fields based on product type or category

### 📊 Admin Enhancements
- Add custom columns to orders and products list
- Filter orders by custom meta fields
- Export order data with custom fields included

### 🔒 Security & Spam
- Block fake account registrations
- Rate-limit add-to-cart actions
- Disable REST API endpoints not in use

---

## How I Use These

Each snippet is pulled from real client projects and tested across 
WooCommerce versions. I treat these as a reference library — 
not a plug-and-play drop-in, so always review before using 
in your own environment.

---

## Compatibility

- WooCommerce 6.x – 8.x
- WordPress 6.x
- Tested on PHP 8.0 and 8.1
